# Activation

Activation include l&#39;ultimo prodotto di gestione tag denominato Launch, di Adobe, Dynamic Tag Management (DTM) e Triggers.

## Experience Platform Launch {#section_05DF74E9EAC94518A155095C65F54568}

Distribuisci e gestisci i tag di analisi, marketing e pubblicità necessari per sviluppare esperienze cliente rilevanti.

Consulta [Launch, di Adobe](https://marketing.adobe.com/resources/help/en_US/experience-cloud/launch/).

## Experience Cloud Triggers {#section_EEA3A8B5EFCB4F8BABC4ADAF3DD41831}

## Debugger di Experience Cloud {#section_682FCF8A22ED47D0ACF94204AB94B444}

L&#39;estensione di debugger di Adobe Experience Cloud per Chrome esamina le pagine Web e aiuta a trovare problemi con l&#39;implementazione delle soluzioni Experience Cloud.

Per assistenza, consulta [Experience Cloud Debugger](https://marketing.adobe.com/resources/help/en_US/experience-cloud-debugger/) .
